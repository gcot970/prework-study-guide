HTML

  • Hypertext Markup Language (HTML) is a programming language that gives the content structure.
  • The head element contains information (or metadata) about the webpage.
  • The body element represents the visible content shown to the user.
  • A web application is an application that is accessed via a web browser and allows users to perform tasks on the Internet.
  • HTML is made up of elements. Common elements include:
    • Content elements:
    • Text elements
    • Image elements
    • Header elements
    • List elements
    Elements that divide and describes a webpage:
    • Meta-related elements
    • Content division elements
    • Semantic elements
    Semantic Elements: the name of an element that describes its content and the role it plays on a webpage
  • An opening tag and closing tag are often represented by angle brackets; used to distinguish itself as an element, where the element begins, and where it ends.
  • A tag refers to only to what appears inside the angle brackets

CSS

  • Cascading Style Sheets (CSS) is a programming language that gives the content style; define how webpage content is presented to users.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds (-A, all) changes in the current branch to the staging area
  • git commit: Commits staged files
  • git pull origin main: Syncs (pulls) our local repo with the base branch (main) in GitHub (origin)
  • git push origin branch-name: Pushes our local branch (branch-name) to GitHub (origin)

JavaScript

  • JavaScript is a programming language that handles logic and allows users to interact with a website.
